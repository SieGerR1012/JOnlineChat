# Сетевой консольный чат на Java

JOnlineChat — проект, реализующий многопользовательский сетевой чат с использованием Java Socket API.  
Проект демонстрирует работу клиент-серверной архитектуры, многопоточности, логирования и базовых принципов проектирования приложений.

## Работа проекта:
### Сервер:
- Поддержка множества клиентов одновременно
- Рассылка сообщений всем подключённым пользователям
- Обработка подключения и отключения клиентов
- Логирование всех событий в файл `file.log`
- Поддержка команд пользователей (например `/exit`)

### Клиент:
- Подключение к серверу по IP и порту
- Выбор имени пользователя при входе
- Отправка и получение сообщений в реальном времени
- Команда выхода из чата `/exit`
- Логирование всех сообщений в файл `file.log`

## Технологии:
- Java 17+
- Socket API (TCP/IP)
- Многопоточность (Thread)
- Collections (CopyOnWriteArrayList)
- Maven
- Unit-тесты (JUnit 5)
- Работа с файлами (FileWriter, BufferedWriter)
